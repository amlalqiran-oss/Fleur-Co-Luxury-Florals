
# Fleur & Co – Luxury Florals 🌸

A premium, high-end e-commerce landing page for a luxury floral boutique. **Fleur & Co** blends elegant typography with a minimalist aesthetic to showcase curated floral arrangements and artisanal bouquets.


[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen)](https://amlalqiran-oss.github.io/Fleur-Co-Luxury-Florals/)

## ✨ Key Features

- **Luxury Brand Aesthetic:** A sophisticated UI designed with a "Paris-inspired" minimalist theme, featuring elegant serif typography and high-quality imagery.
- **Curated Collections:** Specialized sections for Bestsellers, New Arrivals, and Wedding Florals with interactive hover states.
- **Interactive "How It Works" Section:** A clean, step-by-step guide from garden selection to doorstep delivery.
- **Social Proof Integration:** A dedicated testimonials slider featuring global client reviews.
- **Fully Responsive:** Seamlessly optimized for mobile, tablet, and desktop viewing.
- **Newsletter Subscription:** Integrated call-to-action for seasonal inspiration and exclusive access.

## 🛠️ Built With

- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom styling focusing on luxury layouts, modern flexbox/grid systems, and fluid animations.
- **JavaScript:** For interactive elements, smooth scrolling, and mobile navigation toggles.
- **Google Fonts:** Utilizing premium-feel typography (Playfair Display & Montserrat style pairings).

## 📂 Project Structure
